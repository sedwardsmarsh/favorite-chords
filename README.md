# These are some of my favorite chord progressions

## **Circlular nodes are chords that start songs**

### Songlist

- Jack Johnson - Sitting Waiting Wishing
- Ashley Eriksson - Island Song
- Radiohead - Creep

```mermaid
flowchart LR;
    %% All starting chords will have a circle shape

    %% Jack Johnson - Sitting Waiting Wishing
    Am((Am))-->Am7
    Am7-->G
    G-->G7
    G7-->F
    F-->F7
    F7-->C
    C-->C7
    C7-->E
    E-->E7
    E7-->Am
    
    %% Ashley Eriksson - Island Song
    A((A))-->D
    D-->G
    G-->Bm
    D-->A
    A-->G
    Bm-->G
    G-->D
    G-->A
    A-->D
    D-->F#7
    F#7-->G
    G-->E7
    E7-->A

    %% Radiohead - Creep
    G((G))-->B
    B-->C
    C-->Cm
    Cm-->Cm
    Cm-->G
```
